# Roots

A open source 2D narrative engine in the browser.

## Installation

Use [npm](https://www.npmjs.com/) or [yarn](https://classic.yarnpkg.com/en/) to install Roots.

```bash
npm install
```

or

```bash
yarn install
```

## Usage

To start Roots run these scripts in the root folder.

```bash
npm start
```

or

```bash
yarn start
```
